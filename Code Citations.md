## **Step 1: Update assetlinks.json**

Edit [.well-known/assetlinks.json](.well-known/assetlinks.json) and replace:
- `com.yourcompany.yourapp` → Your actual Android package name
- `AA:BB:CC:...` → Your app's **SHA256 certificate fingerprint**

**To get your SHA256 fingerprint:**
```bash
# For debug key
keytool -list -v -keystore ~/.android/debug.keystore -alias androiddebugkey -storepass android -keypass android

# For release key
keytool -list -v -keystore your_keystore.jks -alias your_alias
```
